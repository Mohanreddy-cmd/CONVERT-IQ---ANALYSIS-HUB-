PROJECT DESCRIPTION
Most e-commerce founders are drowning in transactional data, yet completely starved of actual growth insights. 

Shopify, Stripe, and WooCommerce spreadsheet exports are massive, messy, and hard to read. To compute a simple checkout funnel leakage (in dollar value) or a monthly Loyalty Cohort Matrix, you either need to write complex SQL/Python queries or wrestle with fragile Excel macros.

That’s why I built ConvertIQ — an autonomous Customer Lifecycle Ingestion, Mathematical Analytics & Automated Validation Engine. 

With ConvertIQ, anyone can upload raw e-commerce logs and instantly unlock:
🟢 Double-Helix Math Engine: Computes conversion rates, AOV, cart abandonment, and customer loyalty return rates client-side with zero latency.
🟢 Bento-Box Automated EDA: Visualizes geographic sales performance, device-wise conversions, marketing ROI quotients, and checkout drop-off stages out of the box.
🟢 Deep Interactive Research: Run Apriori Market Basket Association (finding purchase affinities with adjustable support/confidence), RFM Segment Matrices, and Seasonal Linear Forecasting.
🟢 Dynamic Executive Reports: Instantly compiles everything—including your active AI Q&A chat history—into stylized corporate PDFs, Microsoft Word (.doc), or multi-sheet Excel workbooks.

No manual schema mapping. No raw data leaks to the cloud ($0 database overhead). Just actionable, executive-ready growth blueprints in seconds. ⚙️

Key Features of the Application
ConvertIQ is designed as a zero-code, single-session, fully automated customer lifecycle intelligence hub. Here is what makes the application powerful:
📥 A. Intelligent Fuzzy Schema Ingest Engine
Auto-Header Normalization: An algorithmic fuzzy-matching layer imports files (CSV or Excel) and automatically binds varying columns (e.g., uid, Customer_Id, userkey) to core unified metrics.
Integrity Audit Check: Performs schema health checks on raw files, flagging null fields, format mismatches, and calculating row completeness.
Instant Dataset Previews: Displays interactive Head (Top 5) and Tail (Bottom 5) rows of your live uploaded database to verify integrity on the spot.
📊 B. Interactive Corporate Dashboard
Dynamic Slicing: Instantly filters your entire transactional database by customized Date Ranges and Device Channels.
Strategic Priority Advice: Synthesizes multi-page datasets into immediate, human-readable directives based on your strategic priority (e.g., Maximize Mobile Conversion, Recover Cart Drops).
🍱 C. Automated EDA Bento Hub (Exploratory Data Analysis)
🌍 Geographic Regional Map Grid: Pinpoints coordinates representing local sales shares and region-specific conversion indexes.
💻 Device Hardware Attribution: Attributes visitor traffic, orders, and total net sales to Desktop, Mobile, and Tablet hardware channels.
💰 Marketing ROI & ROAS Tracker: Calculates marketing channel acquisition cost metrics, tracking return multiplier indexes.
⚠️ Checkout Funnel Leakage Monitor: Calculates conversion progression drop-offs and quantifies the exact financial revenue lost in each stage.
👥 VIP Cohort Retention Matrix: Builds an interactive customer loyalty matrix tracking monthly cohort behaviors over time.
🧪 D. Active Custom Research Lab (Mathematical Algorithms)
🛒 Apriori Market Basket Analysis: Calculates dynamic purchase affinity matrices to identify what products are bought together, featuring adjustable Support and Confidence sliders.
💎 RFM Segmentation Matrix: Ranks your database into distinct customer tier cohorts based on Recency, Frequency, and Monetary scores.
📈 Linear Seasonal Forecasting: Computes linear trend regressions across monthly parameters, predicting future revenues.
💬 E. Dynamic Session AI Co-Pilot
Contextual Grounding: Features a floating chatbot that retains the memory of your active filters and parsed metrics.
Dialogue Retention: Automatically logs every natural language question and formula calculation processed during your session, appending it directly into Section F of your executive report!
📥 F. Multi-Format Corporate Report Hub
🎨 Corporate Multi-Page PDF Compiler: Generates formatted, multi-page PDFs utilizing custom themes (Modern Cyan, Indigo Classic, Slate Mono) featuring visual headers, page indicators, and corporate abstracts.
📝 Microsoft Word Summary (.doc): Compiles fully detailed, executive-ready textual descriptions and dataset summaries perfect for formal presentations.
Excel Workbook Master: Compiles multi-page calculations, trends, and records across individual tabs in a single downloaded spreadsheet file.

Frontend Architecture
React 19 & TypeScript: Provides a type-safe single-page reactive view with declarative state tracking.
Tailwind CSS v4: Utility-driven layout styling.
Motion: Coordinates screen cross-fades and micro-interactions.
Recharts: Builds responsive line, bar, area, and customized checkout funnel vectors.
jsPDF: Generates professional multi-page PDFs directly inside the client browser.
XLSX (SheetJS): Processes the multi-tab spreadsheet compilation.
PapaParse: Handles raw browser-side CSV delimiters and data-type normalization.
Backend Pipeline
Node.js: The underlying JavaScript runtime engine.
Express: Serves the lightweight API bridge (/api/generate-analytics).
@google/genai SDK: Manages server-side, contextually grounded LLM prompts through Google Gemini, maintaining strict key safety.
